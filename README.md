# Star wars API

This project is designed to display character information from star wars

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Abeke02/swapi.git
    ```

2. Go to the project directory:

    ```bash
    cd swapi
    ```

## Usage

```bash
python3 app.py
```
